# StoRê Sport — Landing Page

Landing page editorial para loja de moda esportiva, desenvolvida com HTML, CSS e JavaScript puros, hospedada via GitHub Pages.

🔗 **[Ver projeto ao vivo](https://nunes2705.github.io/landingpage-store-sport/)**

---

## Sobre o projeto

Este projeto é uma landing page completa para a **StoRê Sport**, loja de moda esportiva e casual com curadoria premium em Porto Alegre, RS. O design segue uma estética editorial moderna, com paleta creme e terracota, tipografia serifada e animações suaves de scroll.

Desenvolvido como parte do portfólio de **web design freelancer** da criadora do projeto.

---

## Visual e conceito 

- 🤎 Paleta de cores: creme, areia quente e terracota suave  
- Tipografia: Cormorant Garamond (títulos) e DM Sans (corpo)
- Estética editorial de moda, clean e delicada
- Totalmente responsiva para mobile e desktop

---

## Funcionalidades

- Hero section com layout em duas colunas
- Barra de destaques (parcelamento, proteção UV, tecnologia)
- Seção de tecnologia Rala Bela com cards de benefícios
- Vitrine de produtos organizada por categoria (blusas, jaquetas, leggings)
- Modal de zoom com lupa ao passar o mouse
- Animações de entrada com IntersectionObserver
- Seção de contato com formulário e links diretos para WhatsApp e Instagram
- Rodapé com links sociais

---

## Tecnologias utilizadas

- HTML5 semântico
- CSS3 puro (variáveis, grid, flexbox, animações)
- ⚡ JavaScript vanilla (IntersectionObserver, modal, zoom)
- Google Fonts (Cormorant Garamond, DM Sans)
- Font Awesome 6 (ícones)
- 🚀 GitHub Pages (hospedagem)
